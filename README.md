# Rifat's Portfolio

A modern, responsive portfolio website built with React, Vite, and Tailwind CSS.

## Features

- Modern dark theme design with gradient accents
- Fully responsive layout (mobile, tablet, desktop)
- Smooth interactions and hover effects
- Clean component-based architecture
- Fast performance with Vite

## Tech Stack

- **React 18** - UI library
- **Vite** - Build tool and dev server
- **Tailwind CSS** - Utility-first CSS framework
- **JavaScript** - Programming language

## Getting Started

### Prerequisites

- Node.js 16+ and npm

### Installation

```bash
# Install dependencies
npm install
```

### Development

```bash
# Start the dev server
npm run dev
```

The app will open at `http://localhost:5173`

### Build

```bash
# Create production build
npm run build
```

### Preview

```bash
# Preview production build locally
npm run preview
```

## Project Structure

```
src/
├── main.jsx          # Entry point
├── App.jsx           # Main App component
├── Portfolio.jsx     # Portfolio component
└── index.css         # Tailwind CSS imports
```

## Content Sections

- **Hero Section** - Introduction and contact information
- **Education** - Academic background
- **Experience** - Work history and achievements
- **Skills** - Technical and professional competencies
- **Projects** - Notable projects and accomplishments
- **Call to Action** - Contact prompt

## Customization

Edit `src/Portfolio.jsx` to update:
- Personal information
- Experience details
- Skills and categories
- Projects list
- Contact information
